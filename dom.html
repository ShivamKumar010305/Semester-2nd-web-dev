Document Object Model

The Document Object Model (DOM) is a programming interface (API) for HTML and XML documents, representing the page as a tree of objects that JavaScript can access and manipulate. MDN Web Docs MDN Web Docs +1 Essentially, the DOM acts as a bridge between the static web page and the dynamic functionality provided by JavaScript. When a browser loads a webpage, it creates this tree structure in memory, allowing JavaScript to change the structure, style, and content dynamically without reloading the page
